# Core

Runtime layers for Maataa OS.

Current status:

- `kernel/`: future location for the Rust embedded kernel.
- `renderer/`: planned Blink/CEF renderer layer.
- `node-bridge/`: planned Node.js bridge for filesystem, IPC, and host services.
- `java-layer/`: optional JVM integration layer.
- `python-ml/`: local machine-learning service layer.
- `wasm-layers/`: browser/runtime WASM modules.
